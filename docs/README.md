# CodeBuddy Remote 文档入口

本目录保存当前方案、实现约束和必要的探索材料。

## 推荐读取顺序

1. `codebuddy-remote-final-plan.md`
2. `security-and-pairing-flow.md`
3. `blog/codebuddy-remote-architecture.md`
4. `decision-maps/`
5. `archive/README.md`

## 文档说明

- `codebuddy-remote-final-plan.md`：当前权威产品和技术方案。
- `security-and-pairing-flow.md`：安全模型、扫码绑定、Relay 登录流程和后续安全任务。
- `blog/codebuddy-remote-architecture.md`：面向读者的架构和实现介绍，带演示截图。
- `decision-maps/`：未完成的多步架构决策地图，按 ticket 推进。
- `archive/`：精简探索材料，只在追溯可行性或复跑探针时读取。

## 维护原则

- 做方案、实现和排期时，优先读本目录下的权威文档。
- `archive/` 只保留探索结论和可复跑探针，不作为当前方案来源。
- 新增 Mac Host、安全、协议、客户端、Relay 相关决策时，优先更新对应文档，避免散落在临时报告里。
